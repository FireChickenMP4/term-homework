路由注册

路由按领域拆分为三个文件，由 routes.cpp 统一注册
Routes::registerRoutes(app)               // routes.cpp
 ├─ GET /api/health                         // 健康检查
 ├─ registerBookRoutes(app)                // /api/books/*
    ├─ GET /api/books                       // 列表 + 分页 + 搜索
    ├─ GET /api/books/{id}                      // 详情
    └─ POST/PUT/DELETE                       // [AuthFilter]
 ├─ registerUserRoutes(app)                // /api/login, /api/users/*
    ├─ POST /api/login & /api/register          // 无需鉴权
    ├─ GET /api/me                           // [AuthFilter]
    └─ GET /api/users                         // [AdminFilter]
 ├─ registerLibRoutes(app)                  // /api/borrow, /api/return
 └─ SPA fallback                             // GET /login, /register, /*

核心代码 · Controller 层

Drogon 支持 lambda 直接注册路由，所有 controller 使用同一闭包模板
// 三种变体，结构始终：解析参数 → 调 Model → 组响应 → callback

// 1. GET 查询列表 — 参数 → Model → JSON 数组 + 分页
app.registerHandler("/api/books",
  [&](req, cb) {
    auto page = req->getParameter("page");
    auto books = Book::getAll(page, limit);
    cb(newHttpJsonResponse(json));
  }, {Get});

// 2. POST 新增 — JSON body → Model → 201
app.registerHandler("/api/books",
  [&](req, cb) {
    auto json = req->getJsonObject();
    auto id = Book::add(...);
    resp->setStatusCode(k201Created); cb(resp);
  }, {Post, "AuthFilter"});

// 3. DELETE — 路径参数 + try-catch
app.registerHandler("/api/books/{id}",
  [&](req, cb, int id) {
    try { Book::remove(id); cb(resp); }
    catch (const exception &e) { cb(err(e.what())); }
  }, {Delete, "AuthFilter"});

鉴权系统 · Filter & JWT

Filter 三级鉴权
Filter校验条件适用路由
AuthFilterToken 合法即可/api/me, /api/borrow, /api/books
AdminFilterToken + adminGET /api/users, /api/admin/borrowed
SuperAdminFilterToken + admin + user_id==1内部直接判断 id
鉴权完整链路
客户端登录 → jwt::create() → 返回 Token
后续请求 → Authorization: Bearer <token>
  → AuthFilter::doFilter() → jwt::verify()
  → 注入 payload → fccb() 放行 → Handler
jwt::create() — 签发
header["alg"] = "HS256";
header["typ"] = "JWT";
// 设置 payload
payload["user_id"] = userId;
payload["permission"] = perm;
payload["exp"] = now + 86400*7; // 7 天

// base64url 编码头部和载荷
auto b64h = b64urlEncode(writeString(header));
auto b64p = b64urlEncode(writeString(payload));

// HMAC-SHA256 签名
auto sig = b64urlEncode(hmacSha256(b64h+"."+b64p, SECRET()));
return b64h + "." + b64p + "." + sig;
jwt::verify() — 校验
// 1. 分割 token
auto b64h = token[..dot1];
auto b64p = token[dot1+1 .. dot2];
auto sig = token[dot2+1 ..];

// 2. 重算签名
if (b64urlEncode(hmacSha256(...)) != sig)
  return {};

// 3. 检查过期
if (payload["exp"] < time(nullptr))
  return {};

return payload;
